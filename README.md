# car-data-analytics-dashboard
This repository contains a Power BI dashboard project created primarily for learning and practicing data analytics concepts. The project uses an automobile dataset to explore data cleaning, visualization design, KPI creation, and interactive dashboard development in Power BI.

The goal of this project was to gain hands-on experience with building analytical dashboards, understanding vehicle data, and improving data storytelling skills. It is a learning project and does not aim to provide production-grade analytics or machine learning solutions.
